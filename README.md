# swat-yars

[swat](https://github.com/melezhik/swat) black box testing for yars restapi


# install

    carton

# run tests

    # run tests against default host - 127.0.0.1:9001

    carton exec swat

# run tests against a dedicated host

    carton exec swat 192.168.0.1:9001
